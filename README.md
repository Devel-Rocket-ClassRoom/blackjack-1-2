# 미니 게임 과제: 블랙잭-1 (Blackjack)

## 과제 개요

콘솔에서 동작하는 **블랙잭 게임**을 구현하세요.

배포된 실행 파일(`Blackjack-1.exe`)을 실행하여 완성된 게임을 확인할 수 있습니다.

---

## 과제 방식: 페어 프로그래밍

이 과제는 **2인 1팀** 으로 수행합니다.

### 역할

| 역할 | 담당 | 설명 |
|------|------|------|
| **드라이버** | 프로그래밍이 익숙하지 않은 학생 | 키보드를 직접 조작하여 코드를 입력함 |
| **네비게이터** | 프로그래밍이 더 능숙한 학생 | 방향을 안내하고 코드를 검토함 |

### 팀 구성

| 팀 | 네비게이터 | 드라이버 |
|----|-----------|---------|
| 1 | 이현진 | 이현석 |
| 2 | 손주환 | 김성수 |
| 3 | 김지후 | 김지해 |
| 4 | 남현우 | 이하늘 |
| 5 | 박우민 | 유민준 |
| 6 | 허기영 | 김준영 |
| 7 | 이재서 | 김선진 |
| 8 | 조강현 | 김지훈 |
| 9 | 정호종 | 남상준 |
| 10 | 김광환 | 윤지수 |
| 11 | 최진우 | 안진우 |
| 12 | 김진아 | 주영찬 |
| 13 | 김유찬 | 나상욱 |

### 규칙

- 드라이버와 네비게이터는 **하나의 컴퓨터**를 공유합니다.
- **드라이버만** 키보드를 사용합니다. 네비게이터는 키보드를 건드리지 않습니다.
- 네비게이터는 코드를 **말로만** 안내합니다. 직접 타이핑하거나 마우스를 조작하지 않습니다.
- 네비게이터는 드라이버가 막혔을 때 힌트를 줄 수 있지만, 답을 직접 알려주지 않습니다.
- 역할은 **바꾸지 않습니다.**

## 제출 방법

소스 코드와 별개로 **노션 과제 페이지**에 다음을 작성하여 제출합니다.

### 1. 사전 설계 (코딩 시작 전 작성)

코딩을 시작하기 전, **두 사람이 함께 의논**하여 다음을 결정하고 문서화합니다.

- **작업 순서**: 어떤 순서로 구현할지 단계별 목록으로 작성
- **설계 노트**: 구조를 구상하면서 작성한 노트, 다이어그램, 메모 등 (손으로 그린 것을 사진 찍어 첨부해도 됨)

### 2. 소스 코드

GitHub Classroom 리포지토리에 완성된 코드를 제출합니다.

## 게임 규칙

- 플레이어와 딜러(컴퓨터)가 카드를 받아 **21에 가장 가까운 점수**를 만듭니다.
- 카드 값: A(1 또는 11), K/Q/J(10), 2~10(숫자 그대로)
- 처음에 2장씩 받습니다. 딜러의 첫 번째 카드는 숨깁니다.
- 플레이어가 먼저 **Hit(카드 받기)** 또는 **Stand(멈추기)**를 선택합니다.
- 21을 초과하면 즉시 패배합니다 (**Bust**).
- 딜러는 17 이상이 될 때까지 카드를 받아야 합니다.

---

## 실행 예시

```
=== 블랙잭 게임 ===

카드를 섞는 중...

=== 초기 패 ===
딜러의 패: [??] [♠K]
딜러 점수: ?

플레이어의 패: [♥7] [♦5]
플레이어 점수: 12

H(Hit) 또는 S(Stand)를 선택하세요: H

플레이어가 카드를 받았습니다: [♣9]
플레이어의 패: [♥7] [♦5] [♣9]
플레이어 점수: 21

H(Hit) 또는 S(Stand)를 선택하세요: S

플레이어가 Stand를 선택했습니다.

딜러의 숨겨진 카드: [♠6]
딜러의 패: [♠6] [♠K]
딜러 점수: 16

딜러가 카드를 받습니다: [♦3]
딜러의 패: [♠6] [♠K] [♦3]
딜러 점수: 19

=== 게임 결과 ===
플레이어: 21점
딜러: 19점

플레이어 승리!

새 게임을 하시겠습니까? (Y/N): N
게임을 종료합니다.
```

## 추가 도전 (선택) ★★★

기본 게임을 완성했다면 **베팅 시스템**을 추가하세요.

**규칙:**

- 시작 칩: 1000개
- 매 게임 시작 전 베팅 금액을 입력받습니다.
  - 보유 칩 이하의 금액만 입력 가능합니다.
- 승리 시: 베팅 금액의 2배를 획득합니다.
- 패배 시: 베팅 금액을 잃습니다.
- 무승부 시: 베팅 금액을 돌려받습니다.
- 칩이 0이 되면 게임 오버로 종료합니다.

**실행 예시:**

```
=== 블랙잭 게임 ===
보유 칩: 1000개

베팅 금액을 입력하세요: 200

카드를 섞는 중...

=== 초기 패 ===
...

=== 게임 결과 ===
플레이어: 19점
딜러: 16점

플레이어 승리! (+200개)
보유 칩: 1200개

새 게임을 하시겠습니까? (Y/N): Y
```
